# Prompt Engineering Assignment

## Overview
This assignment demonstrates the basics of Prompt Engineering using AI tools like ChatGPT. It includes different types of prompting techniques along with sample prompts and generated outputs.

---

## Objectives
- Understand Prompt Engineering concepts
- Learn different prompting techniques
- Generate responses using AI models
- Analyze how prompts influence AI outputs

---

## Topics Covered
- Introduction to Prompt Engineering
- Zero-Shot Prompting
- One-Shot Prompting
- Few-Shot Prompting
- Role-Based Prompting
- Prompt Examples and Outputs

---

## Tools Used
- ChatGPT
- Google Gemini
- Web Browser
